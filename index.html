Acerca de esta visualización

Este mapa muestra la distribución geográfica de tsunamis registrados históricamente. El tamaño de cada punto representa la altura máxima del tsunami, mientras que el color indica el año en que ocurrió el evento.

Puedes interactuar con el mapa utilizando las siguientes funciones:

  • Zoom con la rueda del ratón o los botones del panel de control
  • Rotación arrastrando el globo terráqueo
  • Filtrado por años usando los controles deslizantes
  • Información detallada al colocar el cursor sobre cada punto

Visualización creada con Plotly.js | Datos: Registro histórico de tsunamis
